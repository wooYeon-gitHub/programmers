#**배열 만들기(1)**
[문제 바로가기](https://school.programmers.co.kr/learn/courses/30/lessons/181901)

**문제 설명**

> 정수 n과 k가 주어졌을 때, 1 이상 n이하의 정수 중에서 k의 배수를 오름차순으로 저장한 배열을 return 하는 solution 함수를 완성해 주세요.

**입출력의 예**
| n | k | result
| --- | --- | ---
| 10 | 3| [3, 6, 9]
| 15 | 5| [5, 10, 15]

---

**나의 풀이**

```javascript
function solution(n, k) {
  var answer = [];
  for (i = 1; i <= n; i++) {
    if (i % k === 0) answer.push(i);
  }
  return answer;
}
```

**참고 풀이**

```javascript
function solution(n, k) {
  var answer = [];
  for (let i = k; i <= n; i += k) {
    answer.push(i);
  }
  return answer;
}
```
