나머지 구하기

---

문제설명
각에서 0도 초과 90도 미만은 예각, 90도는 직각, 90도 초과 190도 미만은 둔각 180도는 평각으로 분류합니다. 각 angle이 매개변수로 주어질 때 예각일 때 1, 직각일 때 2, 둔각일 때 3, 평각일 때 4를 return 하도록 solution함수를 완성해주세요.

입출력 예
| angle | result |
| --- | --- | --- |
| 70 | 1 |
| 91 | 3 |
| 180 | 4 |

---

나의 풀이

```javascript
fucntion solution(angle) {
  var answer = 0;

  while(true) {
    if (angle < 90) {
      return 1
      break;
    } else if (angle === 90) {
      return 2
      break;
    } else if (angle === 180) {
      return 4
      break;
    } else if (90 < angle < 180) {
      return 3
      break;
    }
  }
  return answer;
}
```

참고풀이

```javascript
function solution(angle) {
  let answer = 1;

  if (angle >= 90) answer++;
  if (angle > 90) answer++;
  if (angle >= 180) answer++;
  if (angle > 180) answer++;

  return answer;
}
```
