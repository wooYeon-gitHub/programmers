숫자비교하기

---

문제설명
정수 num1과 num2가 매게변수로 주어집니다. 두 수가 같으면 1 다르면 -1을 return하도록 solution 함수를 완성해주세요.

입출력 예
| num1 | num2| result |
| --- | --- | --- |
| 2 | 3 | -1 |
| 11 | 11 | 1 |
| 7 | 99 | -1 |

---

나의 풀이

```javascript
fucntion solution(num1, num2) {
  if (num1 === num2) {
    return (1)
  } else { return(-1) };
}
```

참고할 풀이

```javascript
function solution(num1, num2) {
  var answer = num1 === num2 ? 1 : -1;
  return answer;
}
```

```javascript
function solution(num1, num2) {
  return num1 === num2 ? 1 : -1;
}
```
