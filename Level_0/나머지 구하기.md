나머지 구하기

---

문제설명
정수 num1, num2가 매개변수로 주어질 때, num1를 num2로 나눈 나머지를 return 하도록 solution 함수를 완성해주세요.

입출력 예
| num1 | num2| result |
| --- | --- | --- |
| 3 | 2 | 1 |
| 10 | 5 | 0 |

---

나의 풀이

```javascript
fucntion solution(num1, num2) {
  let answer = num1 % num2;
  return answer;
}
```

참고풀이

```javascript
const solution = (num1, num2) => num1 % num2;
```
