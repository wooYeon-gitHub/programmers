나이출력(출생연도)

---

문제설명
머쓱이는 40살인 선생님이 몇 년도에 태어났는지 궁금해졌습니다. 나이 age가 주어질 때, 2022년을 기준 출생 연도를 return하는 solution 함수를 완성해주세여.

입출력 예
| age | result |
| --- | --- |
| 40 | 1983 |
| 23 | 2000 |

---

나의 풀이

```javascript
fucntion solution(age) {
  const curYear = 2023;
  const born = curYear - age;
  return born;
}
```

참고풀이

```javascript
function solution(age) {
  var answer = 2022 - age + 1;
  return answer;
}
```
