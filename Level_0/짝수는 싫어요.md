---

문제설명
정수 n이 매개변수로 주어질 때, n 이하의 홀수가 오름차순으로 담긴 배열을 result하도록 solution 함수를 완성해주세요.

입출력 예
| n | result |
| --- | --- |
| 10 | [1, 3, 5, 7, 9] |
| 15 | [1, 3, 5, 7, 9, 11, 13, 15] |

---

나의 풀이

```javascript
fucntion solution(n) {
  var answer = [];

  for(let i = 1; i <= n; i++) {
    if (i % 2 === 1) {
      answer.push(i)
    }
  }
  return answer;
}
```

참고풀이

```javascript
function solution(n) {
  const answer = [];
  for (let i = 1; i <= n; i++) {
    if (i % 2 !== 0) {
      answer.push(i);
    }
  }
  return answer;
}
```
