#**약수의 합**
[문제 바로가기](https://school.programmers.co.kr/learn/courses/30/lessons/12928)

**문제설명**

> 정수 n을 입력받아 n의 약수를 모두 더한 값을 리턴하는 함수, solution을 완성해주세요.

**입출력 예**
| n | result |
| --- | --- | --- |
| 12 | 28 |
| 5 | 6 |

---

**나의 풀이**

```javascript
fucntion solution(n) {
  let answer = 0;
  for (let i = 1; i*i<=n; i++>) {
    if (n%i == 0) answer += i +( i != ni ? n/i : 0)
  }
  return answer;
}
```

**참고풀이**

```javascript
function solution(n) {
  var answer = 0;
  let i;
  for (i = 1; i <= Math.sqrt(n); i++) {
    if (!(n % i)) {
      answer += i + n / i;
    }
  }
  i--;
  return i === n / i ? answer - i : answer;
}
```
